# ❤️ Heart Disease Prediction (Task 3 - AI/ML Internship)

## 📌 Project Overview
This project is part of my AI/ML Internship tasks. The goal is to build a machine learning model that predicts whether a person is likely to have heart disease based on medical attributes.

Heart disease prediction is an important real-world healthcare application where machine learning helps in early diagnosis and decision-making.

---

## 🎯 Task Objective
- Understand and analyze the heart disease dataset  
- Perform data preprocessing and cleaning  
- Apply machine learning classification algorithms  
- Evaluate model performance  
- Compare models to select the best one  

---

## 📊 Dataset Used
- Dataset: **Heart Disease Dataset** (`heart.csv`)  
- Type: Structured dataset used for classification  

### Features:
- Age  
- Sex  
- Chest Pain Type (cp)  
- Resting Blood Pressure (trestbps)  
- Cholesterol (chol)  
- Fasting Blood Sugar (fbs)  
- Resting ECG (restecg)  
- Maximum Heart Rate (thalach)  
- Exercise Induced Angina (exang)  
- ST Depression (oldpeak)  
- Slope (slope)  
- Number of Major Vessels (ca)  
- Thalassemia (thal)  

- **Target Variable:**  
  - `0` → No Heart Disease  
  - `1` → Heart Disease Present
