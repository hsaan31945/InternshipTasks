# 🏠 House Price Prediction (AI/ML Project)

## 📌 Project Overview
This project focuses on predicting house prices using machine learning models. The goal is to analyze housing data, apply preprocessing techniques, and build regression models to estimate the **median house value** based on various features.

---

## 🎯 Task Objective
- Perform data preprocessing and cleaning  
- Explore and visualize housing dataset  
- Train multiple regression models  
- Evaluate model performance using standard metrics  
- Compare models to find the best-performing one  

---

## 📊 Dataset Used
- Dataset: **California Housing Dataset** (`housing.csv`)  
- Target Variable: `median_house_value`  

### Features:
- Numerical features (e.g., median income, house age, total rooms, population)  
- Categorical feature: `ocean_proximity`  

---

## ⚙️ Data Preprocessing
- Handled missing values using **SimpleImputer**  
- Scaled numerical features using **StandardScaler**  
- Encoded categorical data using **OneHotEncoder**  
- Built preprocessing pipeline using **ColumnTransformer**  

---

## 🤖 Models Applied
The following regression models were implemented:

- Linear Regression  
- Ridge Regression  
- Lasso Regression  
- Random Forest Regressor  
- Histogram Gradient Boosting Regressor  

---

## 📏 Evaluation Metrics
Models were evaluated using:

- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  
- R² Score (Coefficient of Determination)  

---

## 📈 Key Results & Findings
- Tree-based models like **Random Forest** and **Gradient Boosting** outperformed linear models  
- Linear models provided a good baseline but struggled with non-linear relationships  
- Feature scaling and encoding improved model performance significantly  
- Data distribution showed some skewness affecting predictions  

---

## 📊 Visualization & Analysis
- Distribution plots of target variable  
- Count plots for categorical features  
- Summary statistics of dataset  

---

## 🚀 Conclusion
This project demonstrates a complete machine learning workflow:
- Data preprocessing  
- Feature engineering  
- Model training  
- Performance evaluation  

Ensemble models performed best, making them suitable for real-world housing price prediction.

---

## 🛠️ Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn
